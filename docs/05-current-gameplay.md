# Текущий игровой процесс

## Управление

Клавиатура в Godot Preview:

- `A` / `D`: шагнуть по окружности башни влево/вправо.
- `W` / `S`: сменить дорожку вверх/вниз.
- `Q` / `E`: вращать башню/камеру.

Мышь в Godot Preview:

- зажать левую кнопку и тянуть влево-вправо: вращать башню;
- клик снизу слева: шагнуть по окружности влево;
- клик снизу справа: шагнуть по окружности вправо.

Тач-управление:

- свайп влево-вправо: вращать башню;
- тап снизу слева: шагнуть по окружности влево;
- тап снизу справа: шагнуть по окружности вправо.

## Забег

- Нужно выжить до конца таймера.
- Игрок теперь не закреплен на одном секторе: можно ходить по окружности башни.
- Враги, которые добираются до сектора и дорожки игрока, повреждают реактор.
- Оружие стреляет автоматически по ближайшим угрозам.
- Убийства дают энергию.
- Когда энергия заполняется, забег ставится на паузу и предлагает три улучшения.

## Текущие враги

- Runner: быстрый, мало HP, низкая награда.
- Bulwark: медленнее, больше HP, выше награда, появляется после разогрева забега.

## Текущие улучшения

- Быстрее автоогонь.
- Лучше таргетинг по дорожкам.
- Ремонт прочности реактора.
- Бонус энергии за убийства.
- Быстрее вращение башни.
- Больше максимальная прочность реактора.
