# Текущий игровой процесс

## Управление

Клавиатура в Godot Preview:

- `A` / `D`: шагнуть по реакторной ленте влево/вправо.
- `W` / `S`: сменить дорожку вверх/вниз.
- `Q` / `E`: прокрутить реакторную поверхность.
- `Space`: стабилизирующий импульс вокруг игрока.
- `?` в правом верхнем углу: открыть/скрыть управление и легенду.

Мышь в Godot Preview:

- зажать левую кнопку и тянуть влево-вправо: прокрутить реакторную поверхность;
- правая кнопка мыши: стабилизирующий импульс;
- клик снизу слева: шагнуть по ленте влево;
- клик снизу справа: шагнуть по ленте вправо;
- клик по нижней карточке улучшения: применить апгрейд без остановки забега.

Тач-управление:

- свайп влево-вправо: прокрутить реактор;
- тап снизу слева: шагнуть по ленте влево;
- тап снизу справа: шагнуть по ленте вправо;
- тап по компактной карточке улучшения снизу: применить апгрейд на ходу.

## Забег

- Нужно выжить до конца таймера.
- Игрок ходит по развернутой поверхности реактора: сектора остаются круговыми в логике, но на экране показаны как вертикальная лента.
- Враги, которые добираются до сектора и дорожки игрока, повреждают реактор.
- Аварийные вентили появляются на ленте: чтобы закрыть вентиль, встань на его сектор и дорожку до истечения таймера.
- Если вентиль не закрыть вовремя, реактор получает урон.
- Оружие стреляет автоматически по ближайшим угрозам.
- Убийства и закрытые вентили дают энергию.
- Когда энергия заполняется, появляются две компактные карточки улучшений внизу экрана.
- Выбор улучшения больше не ставит забег на паузу: игрок продолжает двигаться, стрелять, закрывать вентили и использовать импульс.
- Стабилизирующий импульс повреждает врагов и закрывает аварийные вентили рядом с игроком, затем уходит на перезарядку.

## Визуальная модель

- Renderer перешел с псевдо-3D цилиндра на развернутую вертикальную реакторную ленту.
- Лента показывает три дорожки, центральный реакторный сердечник и вертикальный передний шов.
- Прокрутка `Q/E` или мышью сдвигает сектора по горизонтали, будто поверхность оборачивается вокруг башни.
- Радар справа сохраняет круговую модель и показывает, где объекты находятся вокруг реактора.
- Это экспериментальный pivot: цель - сделать реактор понятнее и убрать ощущение странной прозрачной катушки.

## Визуальная легенда

- Верхний HUD стал приборной панелью: время/уровень и три полосы состояния.
- Желтая полоса HUD: энергия до следующего улучшения.
- Зеленая полоса HUD: прочность реактора.
- Синяя полоса HUD: готовность стабилизирующего импульса.
- Управление и легенда скрыты в меню `?`, чтобы не перекрывать игру.
- Фиолетовый знак `!`: аварийный вентиль. Его нужно закрывать движением или импульсом.
- Синий щит: тяжелый враг Bulwark. Он медленнее, но живучее.
- Оранжевый блок: обычный враг Runner.
- Риски на ленте: визуальные метки прокрутки поверхности.
- Желтая вертикальная метка: передний шов поверхности, по которому легче увидеть прокрутку.
- Радар справа: компактный вид сверху. Желтая точка - игрок, фиолетовые точки - аварийные вентили, оранжевые/синие точки - враги.
- Нижние карточки: быстрые улучшения, которые можно выбрать без отдельного меню.
- Золотое свечение снизу: доступен быстрый апгрейд.
- Красная тревожная рамка: прочность реактора опасно низкая.
- Полупрозрачные зоны слева/справа снизу: мобильные зоны шага по окружности.
- Красная вспышка на ленте: реактор получил урон.
- Зеленая вспышка: вентиль закрыт.
- Золотая вспышка: улучшение применено.

## Текущие враги

- Runner: быстрый, мало HP, низкая награда.
- Bulwark: медленнее, больше HP, выше награда, появляется после разогрева забега.

## Текущие улучшения

- Быстрее автоогонь.
- Лучше таргетинг по дорожкам.
- Ремонт прочности реактора.
- Бонус энергии за убийства.
- Быстрее вращение башни.
- Больше максимальная прочность реактора.

## Баланс MVP

- Сейчас на уровень нужно `10` энергии.
- На выбор даются `2` улучшения вместо `3`, чтобы экран не перегружался.
- Если улучшение уже висит на экране, новая пачка не появляется поверх него.
